How to run:
1) run pip install virtualenv in command prompt
2) cd to program folder
3) run env\Scripts\activate
4) run python main.py
5) https://developer.spotify.com/console/get-current-user/
	-click get token
	-check "user-read-private", "user-read-email", "user-read-recently-played"
	-copy token
6)go to settings and enter user ID and token and then click save
7)click create playlist 
	-if error occurs change camera value to 0 or 1 in settings
